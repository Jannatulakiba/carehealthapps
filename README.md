# 🏥 CareHealth Apps

A cross-platform health management application built with Flutter & Firebase — available on **Android** and **Web**.

---

## ✨ Features

- 🏃 **Fitness Tracking** — Track daily workouts and physical activity
- 😴 **Sleep Monitoring** — Log and analyze sleep patterns
- 💊 **Medication Reminders** — Never miss a dose
- 📅 **Appointments** — Schedule and manage doctor appointments
- 💧 **Water Reminder** — Stay hydrated with daily water intake tracking
- 🥗 **Nutrition Screen** — Monitor daily nutrition and diet
- 🔐 **Authentication** — Secure login with Firebase Auth (Email/Password)

---

## 🛠️ Tech Stack

| Layer | Technology |
|---|---|

| Backend | Firebase |

## 🚀 Getting Started

### Prerequisites

- Flutter SDK `>=3.0.0`
- Dart SDK `>=3.0.0`
- Firebase account

### Installation

```bash
# Clone the repository
git clone https://github.com/Jannatulakiba/carehealthapps.git

# Navigate to project
cd carehealthapps

# Install dependencies
flutter pub get

# Run the app
flutter run
```

### Firebase Setup

```bash
# Install FlutterFire CLI
dart pub global activate flutterfire_cli

# Configure Firebase
flutterfire configure
```

---

## 🌐 Web Deployment

```bash
# Build for web
flutter build web

# Deploy to Firebase Hosting
firebase deploy --only hosting
```
